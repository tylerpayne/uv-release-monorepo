# Configuration Reference

All uvr configuration lives in the workspace root `pyproject.toml`.

## Runner matrix

```toml
[tool.uvr.runners]
my-native-pkg = ["ubuntu-latest", "macos-14"]
my-python-pkg = ["ubuntu-latest"]
```

Managed via `uvr workflow runners`. Packages without an entry build on `ubuntu-latest` by default.

## Package filtering

```toml
[tool.uvr.config]
include = ["pkg-alpha", "pkg-beta"]   # optional allowlist
exclude = ["pkg-internal"]            # optional denylist
latest = "my-main-package"            # gets the GitHub "Latest" badge
```

## Workflow file

The release workflow lives at `.github/workflows/release.yml`. Generated by `uvr workflow init`, validated by `uvr workflow validate`, edited directly for hook customizations.
